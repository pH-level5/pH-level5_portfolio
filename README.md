## Technology Snapshot

The site is built using a lightweight, framework-free stack:
- **HTML5** — semantic structure
- **CSS3** — layout, animations, and visual design
- **Vanilla JavaScript (ES6+)** — interactivity and dynamic behavior
- **Canvas API** — animated background effects
- **Font Awesome & Google Fonts** — icons and typography

No backend services, databases, or build tools are used.

## Project Intent

This portfolio is a **personal showcase**, not a template or starter kit.

While the repository is publicly visible, the codebase reflects individual design decisions, experiments, and workflows tailored specifically for this site.

If you’re browsing this repository, it’s best viewed as:
