# Tech Guy Dashboard

A Gran Turismo 7-inspired dashboard profile with custom cursor navigation and keyboard controls.

## 🎮 Features

### Custom Arrow Cursor
- Sleek arrow/triangle pointer with glow effects
- **Smooth animated movement** with easing curves
- Hover state with enhanced brightness on interactive elements
- **Keyboard mode**: Slower, elegant glide between focused elements (480ms)
- **Mouse mode**: Fast, responsive tracking (60ms)
- Scale and pulse animation during keyboard navigation

### Keyboard Navigation (GT7-Style)
- **Arrow Keys**: Navigate between cards (Up/Down/Left/Right)
- **Enter**: Activate/click focused element
- **Tab/Shift+Tab**: Move focus forward/backward
- **Escape**: Clear focus and return to mouse mode
- **Number Keys 1-5**: Quick jump between sections
  - `1` - Profile
  - `2` - Projects
  - `3` - Store (default)
  - `4` - Blog
  - `5` - Support

Cursor automatically jumps to focused elements when using keyboard navigation!

### Dashboard Layout
- **Profile**: About, skills, and tech stack
- **Projects**: Showcase of 3D Creator, Garry's Mod addon, Ch4os Engine, and audio packs
- **Store**: Decorated shop-stand with awning, poles, product display, and counter
- **Blog**: Sample posts with dates and snippets
- **Support**: Donation buttons and contact methods

- Neutral color palette (desaturated blues and grays)

## 🚀 Usage

1. Open `index.html` in a modern browser
2. Use mouse to hover and click, or keyboard to navigate
3. Switch sections with tabs or number keys
4. Navigate cards with arrow keys
5. Press Enter to activate links/buttons

## 📱 Responsive
