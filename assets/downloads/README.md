# Downloads Directory

This directory contains downloadable app releases for the website.

## Structure

- Place compiled app files here (e.g., `my-app-1.0.0.zip`)
- For GitHub Pages: Keep files under 100 MB
- For larger files: Use GitHub Releases or external CDN

## Example

```
assets/downloads/
├── my-app-1.0.0.zip
├── my-app-1.0.1.zip
└── checksums.txt
```

## Best Practices

1. **Version naming**: Use semantic versioning (e.g., `app-name-1.0.0.zip`)
2. **Checksums**: Include SHA-256 checksums for verification
3. **File size**: Keep under 50 MB for GitHub Pages if possible
4. **Alternatives**: For larger files, use GitHub Releases asset URLs
